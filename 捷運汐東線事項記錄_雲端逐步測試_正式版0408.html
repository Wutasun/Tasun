捷運汐東線事項記錄｜雲端逐步測試程式（Tasun v5 正式版）

用途：逐步驗證「版本來源、登入承接、Health、Read、Merge、本機資料、上傳、最終同步判定」是否符合 Tasun v5 全站固定口令總版（正式版）。

本測試程式判定標準

  • 正式版號唯一來源只能是 tasun-version.json / tasun-version-loader.js。
  • 開頁同步若成功，最終狀態必須可判定為 已同步；若仍停在 待同步，視為未完成。
  • 若 Read / Merge 成功，但未完成去重與同步完成標記，仍視為 部分成功、不合格。
  • 若 HTTP 401 / 403，視為登入橋接或雲端授權未完成。
  • 若主控台有 JavaScript 錯誤，必須列入診斷結論。
