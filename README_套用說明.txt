本包已將 tasun-global-core.js 直接套入附件網站檔案。

套用方式：
1. 將壓縮檔全部內容覆蓋到 GitHub Pages 專案對應位置。
2. 保留 tasun-global-core.js、tasun-resources.json、tasun-version.json 與各 HTML 在同一層。
3. 本次採用不改 UI 的安全掛入方式：於各頁 </head> 前加入共用核心載入段。
